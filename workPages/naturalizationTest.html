Project #5: Naturalization Test

Previous Next
Go to site See repo

Technologies Used:

HTML, CSS, Bootstrap, jQuery, and Firebase.

This is a quiz game website that shows 10 sample questions from the United States Naturalization Test.

Each screen is decorated with patriotic colors for UI enhancement. In the first screen, the user enters his/her name and then he/she will answer 10 questions and the results will show once the 120 second timer is done or when the user hits the "Done!" button.

The timer ending or the "Done!" button will send the name entered and answers of each question to firebase as you can see in the last slide in the carousel. So the answers of each user can visible to the quiz maker.

  • Used HTML & CSS to create vivid and patriotic colors for UI enhancement.
  • Used Bootstrap for responsiveness.
  • Uses Firebase to get the options checked and upload them to database to see which questions the players got right and wrong.
