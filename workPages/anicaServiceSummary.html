Technologies Used:

HTML, CSS, Bootstrap, AngularJS.

Customer:

WellConnectMDS, Columbia.

Role:

Front-End Developer

This is a service ticket application that allows users to create a profile through registering their email. Once the profile is created, users can create service tickets for whatever issue they are facing. Users can edit the ticket they created, add files, change priortity, delete the ticket and more.

This was a project made through Angular collaboratively with another developer that lived in Europe. He was responsible for the back end and I was responsible for the UI. I designed the front end structure, styled it with vidid colors, bootstrap tools such as the navbar and made it responsive.

  • Used HTML for front end design of website.
  • Used Bootstrap for styling and responsiveness.
  • Used Angular to create different components for each new page of applicatione.
  • Communicated through Skype to back end team in European time zone.
