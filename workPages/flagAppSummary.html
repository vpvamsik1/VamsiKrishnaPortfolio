Project #5: Flag App

Previous Next
Go to site See repo

Technologies Used:

HTML, CSS and PHP.

This is an application in which when a user types in the name of a country, US state, or the state of another country, the flag of the entity entered will show below.

I developed this using PHP, namely its "explode" feature. Once a country (for example) is entered, the whole page from "https://en.wikipedia.org/wiki/File:Flag_of_" + country entered + ".svg" will show on the page. But by using the "explode" feature I was able to just break it down to what I wanted from the page, which is the flag.

If, for example, a user enters an entity that does not exist, a red alert will show stating "Sorry we could not find [entity entered]" as you can see in the last screenshot.

  • Used HTML & CSS for front end design.
  • Used Bootstrap for responsiveness.
  • Uses PHP to echo flag page and explode the flag from the page.
